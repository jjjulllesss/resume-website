# Resume Website

A modern resume website built with React, Vite, and Tailwind CSS.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Update your information in `data.js`

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
